**Digital-маркетолог в EdTech** — делаю так, чтобы онлайн-школы получали заявки, а не просто красивые страницы.

Работаю с двумя брендами детского образования:

- **Зерокодер** — нейросети и программирование для детей 7–17 лет
- **Матриус** — скорочтение и ментальная арифметика для детей 5–17 лет

## Чем занимаюсь

Строю лендинги, воронки и автоматизации для EdTech. Фокус — на конверсии: от первого клика до оплаты. Считаю CPL, CR1/CR2, ROMI и принимаю решения на данных, а не на ощущениях.

Последний год активно работаю с AI-агентами — перевожу маркетинговые процессы на рельсы автоматизации через Claude Code.

## Проекты

### Zerocoder Landing Factory

Мультиагентная система для генерации высоконверсионных лендингов. 9 специализированных AI-агентов работают конвейером: исследование аудитории, копирайтинг, согласование с заказчиком, вёрстка, тройная экспертиза (критик + дизайнер + пользователь-персона) и финализация. Итерации без лимита — пока результат не будет принят.

**Результат:** от промпта до готового лендинга за ~20 минут вместо 2–5 дней.

### 35+ лендингов

Портфолио из 35+ лендингов для Зерокодера и Матриус — бесплатные уроки, вебинары, диагностики, летние программы. Каждый заточен под свою температуру трафика и возраст аудитории.

## Подход

- Пишу тексты простым языком — для родителей, а не для маркетологов
- Один сильный вариант лучше, чем пять средних
- Каждый лендинг проходит QA-чеклист из 20+ пунктов
- Дизайн-системы для обоих брендов — консистентность без ручного контроля
